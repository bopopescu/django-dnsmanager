Django DNS manager
==================

|License: GPL v3|

This is a DNS manager Django app.

Manual install
--------------

For development in a virtualenv:

.. code:: bash

   $ python3 -m venv venv
   $ source venv/bin/activate
   $ pip3 install -e .
